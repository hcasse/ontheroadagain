Shield moteur

Comme nous l’avons déjà dit, il serait dangereux pour l’Arduino de le relier directement aux moteurs qui demandent bien trop de puissance. A la place, nous allons utiliser un shied dédié (circuit emboîtable sur les broches de l’Arduino). Ce shield, ke L298 MotorChild d’ElecFreaks, fournit : un circuit de puissance permettant d’alimenter jusqu'à 2 moteurs, des commandes permettant de contrôler les moteurs à partir de l’Arduino. La documentation officielle est ici.
